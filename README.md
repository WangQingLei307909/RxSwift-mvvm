# RxSwift-mvvm
一个基于RxSwift + Moya + mvvm架构的简单App（内容不断在丰富、优化中）

app包含了而且生成、识别、扫描、以及RxSwift的使用


RxSwift：
1、Subject
2、Variable
3、doOn
4、Dispose
5、bindto
6、Deiver
7、变化操作
8、Scheduler
......
